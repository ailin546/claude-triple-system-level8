# UI/UX Pro Max — Design Intelligence Database

Searchable reference: 161 color palettes, 57 font pairings, 161 product templates, 99 UX rules, 51+ visual styles, 25 chart types. Knowledge organized into 8 topic subdocs. Use it for data-backed design decisions instead of guessing.

## When to Use
- Starting a new UI: page, dashboard, mobile app, landing page
- Choosing style, palette, typography pairing, or chart type
- Looking up UX rules / a11y checks / interaction patterns
- Generating a persisted design system (`design-system/MASTER.md` + page overrides)

## When NOT to Use
- Critiquing an existing mockup → `design:design-critique`
- WCAG-only audits → `design:accessibility-review`
- Reviewing already-implemented UI → `design-review`
- Pure backend / API / non-visual work
